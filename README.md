# 🛠️ Craft Simulator

A simple but addictive crafting simulator made in Python.

## 🎮 Features
- 🌲 Gather resources (wood, stone, iron, diamond)
- 🛠️ Craft items (planks, sticks, swords)
- 📈 Level up and gain XP
- 🪓 Tool upgrades
- 🎲 Rare drops (diamonds!)
- 🎁 Redeem codes
- 💾 Save system
- 🏆 Achievements

## ▶️ How to Play

### Option 1 (Python installed)
```bash (in terminal)
python main.py
